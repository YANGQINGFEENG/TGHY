# 智慧农业 - 数据上传测试工具

一个功能完整的桌面应用程序，用于测试智慧农业物联网平台的数据上传功能。

## 功能特性

### 核心功能
- ✅ **传感器数据上传**：支持批量上传传感器数据到服务器
- ✅ **文件上传**：支持上传文件到指定接口
- ✅ **定时上传**：可配置上传时间间隔（秒）
- ✅ **上传次数控制**：可配置上传次数
- ✅ **实时状态显示**：显示上传进度、成功/失败次数
- ✅ **详细日志记录**：记录每次上传的详细信息

### 配置功能
- 🎛️ **服务器地址配置**：可自定义服务器地址
- 🎛️ **上传间隔配置**：支持 1-60 秒的时间间隔
- 🎛️ **上传次数配置**：支持 1-100 次的上传次数
- 🎛️ **传感器选择**：可选择要上传的传感器
- 🎛️ **上传模式切换**：支持传感器数据和文件上传两种模式

### UI 特性
- 🎨 **现代化界面**：简洁直观的用户界面
- 🎨 **实时状态更新**：实时显示上传进度和统计信息
- 🎨 **详细日志**：彩色日志显示，便于查看上传状态
- 🎨 **响应式设计**：适配不同屏幕尺寸

## 安装与运行

### 方式一：开发模式运行

1. **安装依赖**
```bash
cd upload-tester
npm install
```

2. **启动应用**
```bash
npm start
```

### 方式二：打包为 EXE 文件

1. **打包应用**
```bash
npm run build:win
```

2. **安装应用**
打包完成后，在 `dist` 文件夹中找到安装程序，双击安装即可。

## 使用说明

### 1. 配置服务器地址
在左侧配置面板中输入服务器地址，默认为 `http://localhost:3000`

### 2. 选择上传模式
- **传感器数据上传**：上传模拟的传感器数据
- **文件上传**：上传指定的文件

### 3. 配置上传参数
- **上传间隔**：设置每次上传之间的时间间隔（秒）
- **上传次数**：设置总共上传的次数

### 4. 选择传感器（传感器模式）
在传感器列表中勾选要上传数据的传感器

### 5. 开始上传
点击"开始上传"按钮，应用将按照配置自动上传数据

### 6. 查看日志
在右侧日志区域可以查看每次上传的详细信息和状态

## 配置文件

应用会自动保存配置到 `config.json` 文件，包括：

## 常见问题

### Q: 上传失败怎么办？
A: 检查以下几点：
- 服务器地址是否正确
- 服务器是否正在运行
- 网络连接是否正常
- 查看日志中的错误信息

### Q: 如何修改默认传感器配置？
A: 编辑 `config-manager.js` 文件中的 `DEFAULT_SENSORS` 数组

### Q: 如何自定义上传接口？
A: 在文件上传模式下，可以自定义上传接口地址
