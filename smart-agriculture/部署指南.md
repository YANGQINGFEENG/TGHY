# smart-agriculture 项目部署指南

## 1. 环境准备

### 1.1 系统要求
- **操作系统**：Windows 10/11、Ubuntu 20.04+、CentOS 7+或macOS 10.15+
- **Node.js**：v18.0.0 或更高版本
- **npm**：v9.0.0 或更高版本
- **数据库**（可选）：MySQL 5.7+ 或 MariaDB 10.0+

### 1.2 安装 Node.js
1. **Windows**：从 [Node.js 官网](https://nodejs.org/) 下载并安装
2. **Linux**：
   ```bash
   # Ubuntu/Debian
   curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
   sudo apt-get install -y nodejs
   
   # CentOS/RHEL
   curl -fsSL https://rpm.nodesource.com/setup_20.x | sudo bash -
   sudo yum install -y nodejs
   ```
3. **验证安装**：
   ```bash
   node -v
   npm -v
   ```

## 2. 项目部署

### 2.1 获取项目代码
1. **克隆仓库**：

## 3. 数据库配置（可选）

### 3.1 安装 MySQL
1. **Windows**：从 [MySQL 官网](https://dev.mysql.com/downloads/) 下载并安装
2. **Linux**：
   ```bash
   # Ubuntu/Debian
   sudo apt-get install mysql-server
   
   # CentOS/RHEL
   sudo yum install mysql-server
   ```

### 3.2 导入数据库结构
1. **启动 MySQL 服务**：
   ```bash
   # Windows
   net start mysql
   
   # Linux
   sudo systemctl start mysql
   ```
2. **登录 MySQL**：
   ```bash
   mysql -u root -p
   ```
3. **导入 SQL 文件**：
   ```bash
   source /path/to/smart-agriculture/database.sql
   ```

### 3.3 配置数据库连接
1. **创建 .env 文件**：
   ```bash
   touch .env.local
   ```
2. **添加数据库连接信息**：
   ```
   DATABASE_URL="mysql://username:password@localhost:3306/smart_agriculture"
   ```

### 3.4 修改 API 路由
- 将模拟数据替换为真实数据库连接
- 示例：
  ```typescript
  // app/api/sensors/route.ts
  import { NextRequest, NextResponse } from 'next/server'
  import mysql from 'mysql2/promise'

  // 创建数据库连接
  const pool = mysql.createPool({
    host: process.env.DB_HOST || 'localhost',
    user: process.env.DB_USER || 'root',
    password: process.env.DB_PASSWORD || '',
    database: process.env.DB_NAME || 'smart_agriculture',
    waitForConnections: true,
    connectionLimit: 10,
    queueLimit: 0
  })

  // GET /api/sensors - 获取所有传感器
  export async function GET(request: NextRequest) {
    try {
      const connection = await pool.getConnection()
      const [rows] = await connection.query('SELECT * FROM sensors')
      connection.release()
      
      return NextResponse.json({
        success: true,
        data: rows,
        total: rows.length,
      })
    } catch (error) {
      return NextResponse.json(
        { success: false, error: '数据库连接失败' },
        { status: 500 }
      )
    }
  }
  ```

## 4. 生产环境部署

### 4.1 使用 PM2 管理进程
1. **安装 PM2**：
   ```bash
   npm install -g pm2
   ```
2. **启动应用**：
   ```bash
   pm2 start npm --name "smart-agriculture" -- start
   ```
3. **设置开机自启**：
   ```bash
   pm2 startup
   pm2 save
   ```

### 4.2 使用 Nginx 作为反向代理
1. **安装 Nginx**：
   ```bash
   # Ubuntu/Debian
   sudo apt-get install nginx
   
   # CentOS/RHEL
   sudo yum install nginx
   ```
2. **配置 Nginx**：
   ```bash
   sudo nano /etc/nginx/sites-available/smart-agriculture
   ```
3. **添加配置**：
   ```nginx
   server {
     listen 80;
     server_name your-domain.com;
     
     location / {
       proxy_pass http://localhost:3000;
       proxy_http_version 1.1;
       proxy_set_header Upgrade $http_upgrade;
       proxy_set_header Connection 'upgrade';
       proxy_set_header Host $host;
       proxy_cache_bypass $http_upgrade;
     }
   }
   ```
4. **启用配置**：
   ```bash
   sudo ln -s /etc/nginx/sites-available/smart-agriculture /etc/nginx/sites-enabled/
   sudo nginx -t
   sudo systemctl restart nginx
   ```

### 4.3 配置 HTTPS（可选）
1. **安装 Certbot**：
   ```bash
   # Ubuntu/Debian
   sudo apt-get install certbot python3-certbot-nginx
   
   # CentOS/RHEL
   sudo yum install certbot python3-certbot-nginx
   ```
2. **获取 SSL 证书**：
   ```bash
   sudo certbot --nginx -d your-domain.com
   ```
3. **验证自动续期**：
   ```bash
   sudo certbot renew --dry-run
   ```

## 6. 常见问题排查

### 6.1 依赖安装失败
- **问题**：npm install 失败
- **解决方案**：
  ```bash
  # 清除缓存
  npm cache clean --force
  # 重新安装
  npm install
  ```

### 6.2 构建失败
- **问题**：npm run build 失败
- **解决方案**：
  - 检查 TypeScript 错误
  - 确保所有依赖已正确安装

### 6.3 服务器启动失败
- **问题**：npm run start 失败
- **解决方案**：
  - 检查端口是否被占用
  - 检查环境变量配置

### 6.4 数据库连接失败
- **问题**：无法连接数据库
- **解决方案**：
  - 检查数据库服务是否运行
  - 检查数据库连接参数是否正确
  - 检查防火墙设置

## 9. 项目功能说明

### 9.1 核心功能
- **传感器监控**：实时监控温度、湿度、光照、土壤湿度等传感器数据
- **数据可视化**：通过图表展示传感器数据趋势
- **设备管理**：查看设备在线状态和电池电量
- **数据分析**：分析传感器数据，提供趋势和异常检测

### 9.2 API 接口
- **GET /api/sensors**：获取所有传感器
- **GET /api/sensors?type=temperature**：按类型获取传感器
- **GET /api/sensors/[id]/data**：获取传感器数据
- **GET /api/sensor-types**：获取传感器类型

### 9.3 前端页面
- **首页**：仪表板，展示所有传感器概览
- **传感器类型页面**：按类型展示传感器数据和图表
- **数据详情页面**：查看单个传感器的详细数据
