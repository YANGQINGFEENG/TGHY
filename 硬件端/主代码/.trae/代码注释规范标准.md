# 代码注释规范标准

## 1. 文件头注释

每个源文件（.c）和头文件（.h）都应该包含文件头注释：

```c
/**
 * @file    文件名.c
 * @brief   文件功能简述
 * @details 文件功能详细描述（可选）
 * @author  作者姓名
 * @date    创建日期
 * @version 版本号
 * @note    注意事项（可选）
 */
```

## 2. 函数注释

所有函数都应该包含函数头注释：

```c
/**
 * @brief   函数功能简述
 * @details 函数功能详细描述（可选）
 * @param   参数名 参数说明
 * @param   参数名 参数说明（多个参数时）
 * @return  返回值说明
 * @retval  特定返回值说明（可选，用于说明特定返回值的含义）
 * @note    注意事项（可选）
 * @example 使用示例（可选）
 */
```

### 示例：

```c
/**
 * @brief   读取DHT11传感器数据
 * @details 从DHT11传感器读取温湿度数据，并进行校验
 * @return  读取结果
 * @retval  0: 成功
 * @retval  1: 失败
 * @note    调用此函数前需要先初始化DHT11
 */
uint8_t DHT11_ReadData(void)
{
    // 函数实现
}
```

## 3. 变量注释

### 3.1 全局变量注释

```c
uint8_t g_wifi_connected = 0; // WiFi连接状态标志（0: 未连接, 1: 已连接）
```

### 3.2 局部变量注释

```c
uint8_t retry = 0; // 重试计数器
```

### 3.3 结构体注释

```c
/**
 * @brief 传感器数据结构体
 */
typedef struct {
    float temperature;  // 温度值（℃）
    float humidity;     // 湿度值（%）
    uint16_t light;     // 光照强度
} SensorData_t;
```

## 4. 宏定义注释

```c
#define MAX_WIFI_RETRY 5        // WiFi连接最大重试次数
#define MAX_SERVER_RETRY 3      // 服务器连接最大重试次数
```

## 5. 代码块注释

### 5.1 功能模块注释

```c
/* ==================== WiFi连接功能 ==================== */

/**
 * @brief 检查WiFi连接状态
 * @return 1: WiFi已连接, 0: WiFi未连接
 */
uint8_t check_wifi_status(void)
{
    // 函数实现
}
```

### 5.2 重要代码段注释

```c
// 发送起始信号
GPIO_ResetBits(DHT11_PORT, DHT11_PIN);
delay_ms(20); // 拉低至少18ms
GPIO_SetBits(DHT11_PORT, DHT11_PIN);
delay_us(30); // 拉高20-40us
```

## 6. 注释掉的代码处理

### 6.1 临时调试代码

使用 `// TODO:` 标记：

```c
// TODO: 临时注释，调试完成后删除
// printf("[DEBUG] 测试信息\r\n");
```

### 6.2 废弃功能代码

使用 `// DEPRECATED:` 标记：

```c
// DEPRECATED: 此功能已废弃，将在下一版本删除
// void old_function(void) { ... }
```

### 6.3 待优化代码

使用 `// FIXME:` 或 `// OPTIMIZE:` 标记：

```c
// FIXME: 此处逻辑有问题，需要修复
// OPTIMIZE: 此处性能可以优化
```

## 7. 注释风格要求

1. **使用中文注释**：根据用户规则，所有注释使用中文
2. **注释内容清晰**：注释应该解释"为什么"而不是"是什么"
3. **保持注释同步**：代码修改时，注释也要同步更新
4. **避免冗余注释**：不要注释显而易见的代码
5. **格式统一**：使用统一的注释格式和缩进

## 8. Doxygen标签说明

| 标签 | 用途 | 示例 |
|------|------|------|
| @file | 文件名 | @file main.c |
| @brief | 简要描述 | @brief 主程序入口 |
| @details | 详细描述 | @details 系统初始化和主循环 |
| @author | 作者 | @author 张三 |
| @date | 日期 | @date 2026-04-11 |
| @version | 版本 | @version 1.0.0 |
| @param | 参数说明 | @param data 数据指针 |
| @return | 返回值说明 | @return 0: 成功, 1: 失败 |
| @retval | 特定返回值 | @retval 0: 成功 |
| @note | 注意事项 | @note 需要先初始化 |
| @warning | 警告信息 | @warning 此函数不可重入 |
| @see | 参见 | @see DHT11_Init() |
| @example | 示例 | @example 使用示例 |

## 9. 代码组织结构

### 9.1 头文件包含顺序

```c
#include "stm32f10x.h"          // 1. 标准库头文件
#include <stdio.h>              // 2. 标准C库头文件
#include "Delay.h"              // 3. 项目公共头文件
#include "Hardware/dht11.h"     // 4. 硬件驱动头文件
```

### 9.2 代码块顺序

```c
// 1. 文件头注释
// 2. 包含头文件
// 3. 宏定义
// 4. 类型定义
// 5. 全局变量
// 6. 函数声明
// 7. 函数实现
```
